#### :large_orange_diamond: About us
我們團隊是一群漫畫重度愛好者，都因為苦於漫畫書單太龐大且觀看的平台太多，書單四散在各處又難以整理跟紀錄，而目前國內也沒有一個能妥善管理的收藏平台，因此我們決定一起開發一個漫畫聚合平台，幫助漫畫同好們能更方便整理與收藏漫畫~ 造福廣大漫迷們就是我們團隊的願景!!!
